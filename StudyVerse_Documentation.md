# StudyVerse Documentation

## What is StudyVerse?

StudyVerse is a comprehensive, AI-powered learning platform designed to enhance the study experience for students and lifelong learners. It combines a modern web application with a browser extension to provide an all-in-one solution for organizing study materials, tracking learning progress, collaborating with peers, and maintaining focus during study sessions.

The platform integrates artificial intelligence to offer personalized assistance, automated summaries, and intelligent recommendations. StudyVerse aims to make learning more efficient, engaging, and productive by centralizing all study-related activities in one intuitive interface.

## Key Features

### 1. Library
**Overview**: A digital repository for organizing and managing all your study resources in one place.

**Detailed Functionality**:
- **Resource Types Supported**: Store notes, PDFs, PowerPoint presentations, websites, videos, and books
- **Organization**: Categorize resources by subject with color-coded tags
- **AI-Powered Chat**: Each resource has an integrated AI assistant that can answer questions about the content, provide explanations, and help with understanding complex topics
- **Note-Taking**: Add personal notes to any resource with highlighting capabilities
- **Search and Filter**: Quickly find resources using subject filters and search functionality
- **Local Storage**: All data is stored locally in your browser for privacy and offline access

**Use Cases**:
- Compile lecture notes from different sources
- Save important research papers and articles
- Organize textbook references and video tutorials
- Create subject-specific resource collections

### 2. Virtual Study
**Overview**: AI-powered learning workspaces that transform passive video watching into active, productive study sessions.

**Detailed Functionality**:
- **YouTube Integration**: Embed and watch educational videos directly within the platform
- **Automatic Summaries**: AI generates concise summaries of video content
- **Smart Note-Taking**: Take structured notes with different formats (headings, bullets, highlights)
- **Time Tracking**: Automatically tracks study time spent on each topic
- **Project-Based Learning**: Create dedicated workspaces for specific study projects
- **AI Chat Assistant**: Get instant answers to questions about the video content
- **Progress Monitoring**: Track time spent and maintain study streaks

**Use Cases**:
- Watch Khan Academy or Coursera videos while taking notes
- Create study sessions for exam preparation
- Review lecture recordings with AI-generated summaries
- Track study time for productivity analysis

### 3. Marketplace
**Overview**: A community-driven platform for discovering and sharing high-quality study resources.

**Detailed Functionality**:
- **Resource Discovery**: Browse curated study materials created by other users
- **Subject Categorization**: Resources are organized by academic subjects
- **Pricing Model**: Mix of free and premium resources (priced in credits)
- **Quality Metrics**: Resources display view counts, likes, and trending status
- **One-Click Addition**: Easily add marketplace resources to your personal library
- **Creator Recognition**: View resource creators and their contributions

**Use Cases**:
- Find comprehensive study guides for difficult subjects
- Access cheat sheets and formula collections
- Discover interactive presentations and tutorials
- Support community creators through premium purchases

### 4. Calendar
**Overview**: A study scheduling tool that helps maintain consistent study habits with smart reminders.

**Detailed Functionality**:
- **Session Planning**: Schedule study sessions with specific topics and durations
- **Visual Calendar**: Monthly view with study sessions clearly marked
- **Reminder System**: Set email and push notifications for upcoming sessions
- **Time Management**: Specify session duration and track planned vs. actual study time
- **Subject Organization**: Color-code sessions by subject for easy identification

**Use Cases**:
- Plan weekly study schedules around classes
- Set exam preparation timelines
- Schedule review sessions for difficult topics
- Maintain consistent study routines

### 5. Post Doubts
**Overview**: A collaborative Q&A platform combining peer learning with AI assistance.

**Detailed Functionality**:
- **Public Q&A**: Post questions publicly to get help from the community
- **AI-First Responses**: Every question gets an immediate AI-generated answer
- **Peer Contributions**: Community members can provide additional answers and explanations
- **Voting System**: Upvote helpful answers to surface the best responses
- **File Attachments**: Attach images or documents to questions for better context
- **Subject Tagging**: Categorize questions by subject for targeted visibility
- **Real-time Updates**: Get notifications when new answers are posted

**Use Cases**:
- Get quick explanations for confusing concepts
- Share complex problem-solving approaches
- Collaborate on homework and assignments
- Build a knowledge base of frequently asked questions

### 6. Analytics
**Overview**: Comprehensive study tracking and productivity analysis powered by a browser extension.

**Detailed Functionality**:
- **Browser Extension**: Tracks website visits and time spent across all tabs
- **Productivity Classification**: Automatically categorizes websites as productive or distracting
- **Time Analytics**: Detailed breakdowns of study time vs. distraction time
- **Domain Insights**: View time spent on educational sites vs. social media
- **Real-time Monitoring**: Live tracking of current browsing activity
- **Productivity Scores**: Calculate focus levels based on website usage patterns

**Use Cases**:
- Identify and reduce time spent on distracting websites
- Track study session effectiveness
- Monitor overall learning productivity
- Set goals for focused study time

### 7. Projects
**Overview**: A showcase platform for student projects and creative work.

**Detailed Functionality**:
- **Project Gallery**: Browse projects created by other students
- **GitHub Integration**: Link to source code repositories
- **Live Demos**: Access deployed project URLs
- **Community Feedback**: Like/dislike system for project ratings
- **Personal Showcase**: Display your own projects and achievements
- **Technology Tags**: Filter projects by programming languages and frameworks

**Use Cases**:
- Get inspiration from other students' projects
- Showcase coding assignments and personal projects
- Learn from open-source implementations
- Build a portfolio of completed work

## Technical Architecture

### Web Application
- Built with Next.js and React for modern web performance
- TypeScript for type safety and better development experience
- Tailwind CSS for responsive, customizable styling
- Local storage for data persistence and privacy
- Framer Motion for smooth animations and transitions

### Browser Extension
- Chrome/Firefox compatible manifest V3 extension
- Background service worker for continuous tracking
- Content scripts for web application integration
- Secure communication between extension and web app

## Privacy & Data Security

- All study data stored locally in browser storage
- No personal data sent to external servers
- Extension only tracks website domains and visit times
- AI interactions processed through secure, privacy-focused APIs
- User content remains private and under user control

## Getting Started

1. Visit the StudyVerse web application
2. Complete the onboarding process to explore features
3. Install the browser extension for analytics tracking
4. Start by adding resources to your Library
5. Create your first Virtual Study session
6. Schedule study sessions in the Calendar
7. Explore the Marketplace for additional resources

StudyVerse is designed to adapt to your learning style and grow with your educational journey, providing tools that make studying more effective, organized, and enjoyable.
